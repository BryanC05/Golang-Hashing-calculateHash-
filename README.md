# Go Blockchain Hash (SHA-256) Demo

This is a simple Go program that demonstrates the **core concept of hashing** in a blockchain. It shows how a block's "fingerprint" is generated by combining all its data and running it through the **SHA-256** algorithm.

The main purpose of this project is to visually demonstrate the **"avalanche effect,"** which shows how a tiny, 1-character change in a block's data results in a completely different hash. This is the fundamental principle that makes a blockchain secure and tamper-evident.

## ðŸ§  Key Concepts

The `calculateHash` function is the heart of this project. It performs the following steps:

1.  **Concatenates Data:** It combines all the block's data (Timestamp, Data, Previous Hash, and Nonce) into a single, long string.
2.  **Uses SHA-256:** It feeds this string into the SHA-256 hashing algorithm.
3.  **Generates a "Fingerprint":** It produces a unique 64-character hexadecimal string. As demonstrated, changing even one character in the input (`Bob` -\> `B0b`) produces a completely different hash, making tampering obvious.
